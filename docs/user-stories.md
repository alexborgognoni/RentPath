# **User Stories**

## 🧑‍💼 **Property Manager User Stories**

### 🎯 Account & Verification

- As a property manager, I want to sign up with my email and password, so that I can manage rental listings.
- As a property manager, I want to indicate if I'm a private landlord or a business, so that the platform can tailor my verification process.
- As a business manager, I want to upload my company documents, so that I can verify my identity and listing rights.
- As a property manager, I want to know if my account is verified, so that I can trust the platform and renters can trust me.

### 🏘️ Property Management

- As a property manager, I want to create and manage property listings, so that I can advertise available rentals.
- As a property manager, I want to add other managers to a property, so that teams or agencies can collaborate on listings.
- As a property manager, I want to make a property invite-only, so that only selected renters can apply.

### 📬 Invitations

- As a property manager, I want to create a general invitation link, so that I can share it publicly or with multiple renters.
- As a property manager, I want to send a direct invitation to a specific email, so that only that renter can apply.
- As a property manager, I want to set usage limits and expiration dates on invitations, so that I can control access.

### 📝 Application Management

- As a property manager, I want to view rental applications submitted to my properties, so that I can review and respond.
- As a property manager, I want to verify renter profiles before accepting them, so that I can reduce risk and fraud.

---

## **Renter User Stories**

### 🧾 Account & Verification

- As a renter, I want to sign up with my email and password, so that I can apply for rental properties.
- As a renter, I want to upload identity, employment, and income documents, so that my profile can be verified.
- As a renter, I want to see my verification status, so that I know when I’m eligible to apply for properties.

### 📬 Invitations

- As a renter, I want to create an account using an invitation link, so that I can apply to a specific property.
- As a renter, I want to apply to a property using a general invite link, so that I don’t need a direct email.

### 📝 Application Process

- As a renter, I want to fill out a rental application with a message, so that I can introduce myself to the property manager.
- As a renter, I want to view the status of my application, so that I know if I've been accepted or rejected.
- As a renter, I want to apply to multiple properties, so that I can increase my chances of finding a home.

---

## ⚙️ Platform & System-Level Stories

### 🔐 Security & Access

- As the system, I want to restrict access to invite-only properties unless a valid token is provided.
- As the system, I want to ensure only verified property managers can verify renters.
- As the system, I want to encrypt and protect uploaded documents, so that user data remains private and secure.
